# 购物中心 / 外卖插件 v4 说明

本版本已按你的反馈调整：

- 手机端底部弹窗布局继续保留。
- 删除弹窗标题下方“手机端适配 · 聊天卡片版”。
- 购物中心 AI 补货 API 预设增加“删除预设”。
- 手动上传商品去掉“款式”和“备注”。
- 手动上传商品里的“详情解释”改为“详情页”。
- 外卖手动上传只保留名称、分类、价格。
- 删除“写入主动购买提示 / 写入主动点单提示”，只保留“弹购买卡片 / 弹外卖卡片”。
- 购物备注、外卖订单备注会显示在聊天卡片中，也会写入消息内容给角色看到。
- 参考 330 小手机方式：在主提示词里长期加入可选的购物/外卖动作格式，让角色在正常触发 AI 时可以按自己意愿主动给 user 买东西或点外卖；不是额外写入临时提示。

测试方式：

```bash
npm install
npm run dev
```

如果同一个文件夹之前已经安装过依赖，通常只需要：

```bash
npm run dev
```
